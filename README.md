# OpenShulker

专为 **Folia** 设计的 Paper 插件，手持潜影盒右键空气即可直接打开 GUI 进行物品交互。

## 使用方式

- **右键空气打开** — 手持潜影盒（主手/副手均可），右键空气即可打开界面
- **右键方块放置** — 对着方块右键为原版放置行为，不受插件影响
- **关闭保存** — 关闭 GUI 后自动将物品写回潜影盒

## 防御机制

| 机制 | 说明 |
|---|---|
| **PDC UUID 精确追踪** | 打开时往物品的 `PersistentDataContainer` 写入随机 UUID，关闭时按 UUID 精准定位写入，杜绝同类型覆盖/槽位错乱 |
| **槽位锁定** | 界面打开期间禁止点击、Shift 移动、数字键交换、拖拽操作改变潜影盒所在槽位 |
| **丢弃/换手拦截** | 打开期间禁止丢弃潜影盒、禁止 F 键切换主副手 |
| **死亡恢复** | 记录死亡位置，关闭界面时扫描附近掉落物按 UUID 找回并写入 |
| **堆叠拒绝** | 数量 >1 的潜影盒拒绝打开，防止整组覆写 |
| **退出保存** | 断线时主动关闭界面触发保存路径，数据不丢 |
| **服务端关闭保护** | `onDisable` 时强制关闭所有打开界面，触发保存 |

## 构建

```bash
gradle build
```

构建产物：`build/libs/OpenShulker-1.0-all.jar`，放入服务端 `plugins/` 目录即可。

## 兼容性

- **Folia** — 完全支持（`folia-supported: true`，无调度器调用）
- **Paper 1.26.1+** — 完全支持
- 零配置，开箱即用
